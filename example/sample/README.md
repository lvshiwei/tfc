This is a very sample example project for how to use tfc

## how to start

```shell
# install
npm install

# generate with default jsonMock
# if only you had install tfc global
# see [How to use](https://github.com/lvshiwei/tfc#get-start)
tfc -j jsonMock.js

# build example project
npm run b

# run dev server
npm run d
```
